# Workspace Manager v0.1.0

首个可用版本。

## 亮点

- 托盘常驻、关闭不退出、最小化到托盘
- 桌面图标显示 / 隐藏
- 任务栏显示 / 隐藏
- 桌面布局保存 / 恢复
- 布局预览图与大图查看
- 模式系统
  - 默认 / 工作 / 演示
  - 自定义模式新建 / 编辑 / 删除
- 开机自启
- 启动最小化到托盘
- 自定义全局快捷键
- 显示主窗口快捷键

## 适用场景

- 演示
- 录屏
- 日常桌面整理前的状态切换
- 恢复被打乱的桌面图标布局

## 当前限制

- 还没有桌面整理规则与自动整理
- 还没有撤销整理与日志导出
- 布局恢复在不同分辨率和缩放环境下仍可能存在偏差
