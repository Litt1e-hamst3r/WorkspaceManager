# Workspace Manager v0.2.0

这是一版围绕日常可用性集中打磨的更新。

如果说 `v0.1.0` 解决的是“能不能用”，那 `v0.2.0` 更像是在回答“能不能顺手用，能不能持续用。

## 亮点

- 主界面重新收紧布局，概览与工作台信息更适合常规窗口尺寸
- 壁纸配置区完成重构，交互更紧凑，图源状态更清晰
- 支持把本地图片加入壁纸图源
- 支持把本地文件夹加入壁纸图源，并参与手动切换与定时轮换
- 壁纸图源支持重选、删除和可用性提示
- 应用主窗口与托盘图标接入项目 logo，并对展示方式做了自适应处理
- README、LICENSE、`.gitignore`、Issue 模板、PR 模板补齐，仓库现在更适合公开维护

## 壁纸能力更新

- 支持三类图源统一管理：
  - 内置在线图源
  - 本地单张图片
  - 本地图片文件夹
- 本地图源会走统一的缓存与转换流程，保证设置壁纸时路径稳定
- 图源列表会标记异常状态，例如文件缺失、目录缺失、目录为空
- 本地图片与本地文件夹支持直接重选，不必删掉再加一遍

## 体验优化

- 左侧工作台头部加入品牌 logo，但不再强行塞进固定方形容器
- 壁纸添加区去掉过长提示文案，按钮区更紧凑
- 图源卡片增加状态标签，异常项更容易被发现
- 托盘图标不再使用系统默认图标，和主界面品牌更统一

## 工程与开源准备

- 新增 `MIT License`
- 重写 `README.md`，补齐定位、功能、运行方式和路线图
- 新增 GitHub Issue 模板：
  - Bug Report
  - Feature Request
- 新增 PR 模板
- 补充更完整的 `.gitignore`

## 兼容性说明

- 运行环境仍为 Windows 10 / 11
- 仍基于 `.NET 8` + `WPF`
- 当前依旧以手工验证为主，自动化测试会在后续继续补齐
